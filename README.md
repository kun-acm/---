# 📚 基础医学概论刷题

> 一个纯前端、零依赖的基础医学刷题 + 知识点学习工具，涵盖 **解剖学 · 生理学 · 组织学** 三门学科。
> 643 道题 + 全部答案解析 + 三学科知识点，双击 `index.html` 即可使用。

---

## ✨ 功能一览

### 刷题

| 功能 | 说明 |
|------|------|
| 学科筛选 | 单一题库模式下切换解剖学 / 生理学 / 组织学 |
| 单一题库 | 按原顺序逐题刷，下拉切换 18 个题库 |
| 全题库乱序 | 643 题全部打乱，每次重新洗牌 |
| 错题练习 | 只练未掌握的错题，做对自动标记"已掌握" |
| 即时判分 | 点击选项立刻显示对错 + 详细解析 |
| 整组判分 | 全部做完统一提交，跳过题计"未答" |
| 断点续做 | 每模式独立记录进度，关闭再打开接着做 |
| 错题本 | 去重、累计错误次数、点击跳转回做 |
| 键盘操作 | `←` `→` 切题，数字键 `1`-`5` 选选项 |

### 知识点学习

| 功能 | 说明 |
|------|------|
| 类目录式浏览 | 左侧学科 + 章节列表，右侧内容面板 |
| 三学科分开 | 解剖学 / 生理学 / 组织学 独立切换 |
| 要点高亮 | ⭐ 必考点（琥珀色）、⚠️ 易错点（红色） |
| 响应式布局 | 移动端自适应 |

## 🚀 使用方法

### 本地使用

双击 `index.html`，用 Chrome / Edge 打开即可。

> 如果进度无法保存，可启动本地服务器：
> ```bash
> python -m http.server 8080
> # 访问 http://localhost:8080/
> ```

## ⚠️ 注意事项

### 答案准确性

所有答案和解析均含 AI 生成成分，**不能保证 100% 正确**，请以教材为准。

### 数据存储

- 答题进度、错题本存储在浏览器 **localStorage**
- **清除浏览器缓存 / 无痕模式 / 换浏览器 / 换电脑** 会导致数据丢失
- 数据仅存本地，不上传任何服务器
