# 人体解剖学刷题

一个纯前端、零依赖的人体解剖学选择题刷题工具。**单个 HTML 文件**，双击即可在浏览器中使用。

---

## 📦 项目结构

```
解剖学刷题/
├── 解剖学刷题.html    # 主程序（唯一文件）
└── README.md          # 本文件
```

---

## 🎯 功能

| 功能 | 说明 |
|------|------|
| **单一题库模式** | 按原顺序逐题刷，下拉切换题库 |
| **全题库乱序模式** | 163 题全部打乱，每次"重新开始"重新洗牌 |
| **错题练习模式** | 只练未掌握的错题，做对自动标记"已掌握" |
| **即时判分** | 点击选项立刻显示对错 + 详细解析 |
| **整组判分** | 全部做完统一提交，跳过题计"未答"不计分 |
| **断点续做** | 每模式独立记录做到第几题，关闭浏览器再打开接着做 |
| **正确率统计** | 总体正确率 + 各题库正确率 + 完成度进度条 |
| **错题本** | 去重、累计错误次数、做对标记已掌握、点击跳转 |
| **键盘操作** | `←` `→` 切换题目，数字键 `1`-`4` 选选项 |
| **进度重置** | 重置进度 / 清空错题本，均需二次确认 |

---

## 🚀 使用方法

### 打开方式

1. **双击** `解剖学刷题.html`，用浏览器打开即可。
2. 推荐使用 **Chrome** 或 **Edge**，不支持 IE。

> ⚠️ 如果用 `file://` 协议双击打开后进度无法保存，可以用以下方式启动本地服务器：
>
> ```bash
> # 在项目目录下执行：
> python -m http.server 8080
> # 然后浏览器访问 http://localhost:8080/解剖学刷题.html
> ```

### 操作说明

| 操作 | 方式 |
|------|------|
| 选择题库 | 顶部下拉框选择 |
| 切换模式 | 顶栏按钮：单一题库 / 全题库乱序 / 错题练习 |
| 切换判分方式 | 即时判分 / 整组判分 |
| 选择答案 | 鼠标点击选项，或按数字键 `1` `2` `3` `4` |
| 翻题 | 点击底部 ← → 按钮，或按键盘 `←` `→`（或 `A` `D`） |
| 重新开始 | 点击"重新开始"按钮 |
| 整组提交 | 整组判分模式下，点击"提交整组判分" |
| 重置进度 | 右侧设置区 → 确认弹窗 |

## ⚠️ 重要提示

### 答案准确性

所有答案和解析均由 AI 生成，**不能保证 100% 正确**。建议：
- 将答案视为"参考答案"而非"标准答案"
- 如有疑问，请以教材为准
- 建议抽检 5~10 题（尤其脉管学和神经章节）核对教材

### 数据存储

- 答题记录、进度、错题本均存储在浏览器的 **localStorage** 中
- **清除浏览器缓存 / 使用无痕模式 / 换浏览器 / 换电脑** 会导致数据丢失
- 数据存储在本地，不会上传到任何服务器

### 题库数据结构

如需自行添加或修改题目，编辑 `<script>` 中的 `QUIZ_BANKS` 数组即可：

```javascript
const QUIZ_BANKS = [
  {
    id: "motion",           // 题库唯一 ID
    name: "运动系统（骨骼、骨连结、肌肉）",  // 显示名称
    questions: [
      {
        type: "single",     // "single" = 单选题, "tf" = 判断题
        stem: "肱骨属：",   // 题干
        options: ["长骨","短骨","扁骨","不规则骨"],  // 选项数组
        answer: 0,          // 正确答案索引（0=A, 1=B, 2=C, 3=D；判断题 0=对, 1=错）
        exp: "肱骨呈长管状……"  // 解析
      },
      // ...更多题目
    ]
  },
  // ...更多题库
];
```
